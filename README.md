# Postman-Testing

## Thông tin

* Môn học: Kiểm thử phần mềm
* Công cụ: Postman
* Website kiểm thử: https://www.youtube.com/

---

## Mục tiêu

* Làm quen với Postman.
* Gửi các HTTP Request.
* Phân tích Response.
* Kiểm tra Header và Status Code.

---

## Môi trường

* Windows 11
* Postman
* Internet

---

## Các bài kiểm thử

### Test 1: GET Homepage

* Method: GET
* URL: https://canvas.phenikaa-uni.edu.vn

**Kết quả**

* Status Code: 200 OK
* Response Time: 1.85s
* Size: 687.9 KB

---

### Test 2: HEAD Request

* Method: HEAD

**Kết quả**

* Status Code: 200 

---

### Test 3: OPTIONS

* Method: OPTIONS

**Kết quả**

* Status Code: 405 Method Not Allowed

---

## Nhận xét

* Website phản hồi ổn định.
* HTTP Status hợp lệ.
* Header đầy đủ.
* Response Time ở mức chấp nhận được.

---

## Kết luận

Đã sử dụng Postman để kiểm thử các phương thức HTTP cơ bản, phân tích phản hồi của máy chủ và lưu lại kết quả dưới dạng báo cáo cùng hình ảnh minh họa.
